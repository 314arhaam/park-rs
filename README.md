# Park

A CLI tool for parquet files, written in Rust #rust #data

```sh
Usage: park <COMMAND>

Commands:
  head  
  info  
  help  Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
```

## Subcommands

### `info`

Get file info

### `head`

Get file head

## Install

```sh
curl -fsSL https://raw.githubusercontent.com/314arhaam/park-rs/main/install.sh | bash
```
